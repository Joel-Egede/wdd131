Tech Learning Hub

Website Planning Document

Site Name

Tech Learning Hub

Tech Learning Hub is an educational website that provides beginner-friendly programming tutorials, technology learning resources, project ideas, and career guidance for aspiring developers.

Reason: The name clearly communicates that the website is a central place where people can learn programming and modern technology skills.

Optional Domain: techlearninghub.dev

Site Purpose

Tech Learning Hub is designed to help beginners and aspiring technology professionals learn programming and essential tech skills through easy-to-follow tutorials, project guides, learning resources, and career advice.

The website will include information about HTML, CSS, JavaScript, Python, cybersecurity, web development, programming projects, recommended learning resources, and practical tips for building a successful career in technology.

Scenarios

  • What programming language should I learn first if I want to become a web developer?
  • Where can I find beginner-friendly projects to improve my coding skills?
  • What free resources can help me learn HTML, CSS, JavaScript, and Python?
  • How can I prepare for a career in software development or cybersecurity?

Color Scheme

  • Dark Blue (#0D47A1) – Used for the header, navigation, headings, and footer.
  • Light Gray (#F5F5F5) – Used for the page background and content sections.
  • White (#FFFFFF) – Used for cards and content containers.
  • Orange (#FF9800) – Used for buttons, links, highlights, and accent elements.

Typography

  • Roboto – Used for the page headings, navigation menu, body text, and footer.
  • Arial (Fallback Font) – Used if the Roboto font is unavailable on the user's device.
